# NYC-Housing-Analysis
# New York Housing Market Analysis

## Objective
This project analyzes the factors influencing rental prices across different boroughs in New York City. The goal is to identify key drivers of rent variation and understand how location impacts housing prices, using Manhattan as the reference category.


## Dataset
- Source: NYC housing dataset (class project dataset)
- Key Variables:
  - Dependent Variable: Rent Price
  - Independent Variables:
    - Borough (categorical)
    - Number of Bedrooms
    - Size (sqft)
    - Location indicators


## Tools
- Excel (data cleaning)
- SPSS (statistical analysis & regression)


## Methodology
1. Data cleaning and preparation in Excel  
2. Exploratory Data Analysis (EDA)  
3. Descriptive statistics (mean, distribution)  
4. Regression analysis using SPSS  
5. Manhattan used as reference category  

---

## Key Findings
- Manhattan has the highest baseline rental prices  
- Other boroughs show significantly lower rent levels  
- Apartment size and number of bedrooms positively impact rent  
- Location is a strong determinant of housing price variation  



## Business Insights
- Location remains the most critical factor in NYC housing pricing  
- Real estate investors should prioritize borough-level differences  
- Rent pricing strategies should account for both size and location
